## Overview

Potea was envisioned as a digital bridge between passionate plant collectors and curated greenery. The goal was to design a responsive and elegant mobile application where users could securely browse, purchase, and track plant orders with ease—all built with scalable, enterprise-ready technology.

## Challenge

- Create a fast, mobile-first shopping experience with smooth state handling.
- Secure user accounts with JWT-based authentication.
- Build a real-time cart and order summary system optimized for usability.
- Ensure seamless coordination between frontend and backend systems.
- Keep the UI lightweight while integrating complex backend logic.
